Profle Page

Technical Description

The profile page uses basic HTML to structure personal information in a clear and user friendly layout.All content is wrapped in the <main> tag, and the content is logically organised by sections.The user's photo, name, description, skills, availability, and contact form are displayed using elements like <img>, <h1>, <p>, <ul>, <table>, and <form>.

CSS can be used to style elements like the .avatar image, progress bars, buttons, and section layout. While spacing and padding guarantee readability, icons and pictures inside lists add visual interest to the volunteering and interest sections.

These structural and styling choices create a simple, professional profile page that shows the user’s skills, interests, and availability while providing a contact form for communication.

Accessibility

  • Alt text is added for all images.
  • The page has high contrast with dark backgrounds and light text for better readability.
  • The layout is responsive, so it works well on both desktop and mobile.
  • The video is muted and looped to avoid distracting users with sound.
  • Navigation is simple and all links are easy to find.

Content Page

Technical Description

To clearly arrange the content into various topics about sustainable seafood, I used simple HTML tags like <section>, <h>, <p>, and <img>.The CSS can be used to style the layout by adding padding and margins so the text and images are spaced out nicely. Rounded corners, soft shadows, and clean fonts help make the sections visually appealing without being too flashy. The images are sized and positioned to fit well with the text, making the page both informative and easy to read.

Challenges and Lessons Learned

While creating the pages, I faced challenges in understanding how to structure content using HTML tags like <section>, <div>, and <main>. styling with CSS was also challenging, particularly when it came to creating a responsive and clean layout. By using resources like W3Schools, I improved my skills and gained confidence in building web pages that are simple, user-friendly, and well-organized

Compliance
