Splash Page

Technical Description

The splash page uses basic HTML and CSS to create a clean, animated intro screen. The <head> section includes a linked stylesheet, favicon, and a meta tag for redirecting to the homepage after 6 seconds. Structurally, <div> containers organize content like the title and team names using <h1> and <p> tags.

CSS uses height: 100vh and Flexbox in .splash-screen to center content, while a background gradient (set via a CSS variable) adds visual appeal. The .splash-text uses absolute positioning and the @keyframes fadeInOut animation to smoothly fade content in and out. The logo is scaled and positioned using translate and scale for balance.

These choices make the splash screen functional, responsive, and visually engaging, serving as a brief intro before redirecting to the main site.

Accessibility

Accessibility was considered by using clear and readable text with proper semantic tags like <h1> for headings and <p> for content, making it easier for screen readers to understand the page structure. The color contrast between the white text and dark background ensures readability for users with visual impairments. Images include alt text for screen reader support, and content is centered and spaced clearly for easier navigation across different screen sizes.
