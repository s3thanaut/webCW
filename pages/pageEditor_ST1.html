Splash Page

Technical Description

The splash page uses basic HTML and CSS to create a clean, animated intro screen. The <head> section includes a linked stylesheet and a meta tag for redirecting to the homepage after 6 seconds. Structurally, <div> containers organize content like the title and team names using <h1> and <p> tags.

CSS uses height: 100vh and Flexbox in .splash-screen to center content, while a background gradient (set via a CSS variable) adds visual appeal. The .splash-text uses absolute positioning and the @keyframes fadeInOut animation to smoothly fade content in and out. The logo is scaled and positioned using translate and scale for balance.

These choices make the splash screen functional, responsive, and visually engaging, serving as a brief intro before redirecting to the main site.

Accessibility

  • Alt text is added for all images.
  • The page has high contrast with dark backgrounds and light text for better readability.
  • The layout is responsive, so it works well on both desktop and mobile.
  • The video is muted and looped to avoid distracting users with sound.
  • Navigation is simple and all links are easy to find.

Volunteer Page

Technical Description

The Volunteer page is built using basic HTML elements and styled with external CSS. A background video is used to make the page more dynamic. Flexbox helps center the content and keep everything aligned. Each volunteer section is placed inside a styled card with hover effects.
