global CSS and Template

Technical Description

The global css (Saved as style.css) contains the all the main stylings used in site throughout all the pages. It also contains the stylings for the header and footer including the navbar stylings.
The same header and footer sections are used in all the pages to maintain consistancy. And in each stylesheet file corresponding to each page, the global stylesheet is imported by using @import "style.css"; line.

background-image: linear-gradient() is used to add the gradient in the background of the header and footer. links are made responsive using the :hover pseudoclass. The css code snippet position: fixed; is responsible for keeping the header with navbar still at the top of the page while scrolling.

These arrangements made the style.css file accessible for all the pages, maintaining the generaly styilng consistant throughout the web site, and keeping the the styling of the header and the footer the same in all pages.

Accessibility

The navigation section is nested in the <header> element, making it appear at the top of the page. And the footer is using the <footer> element for it to appear at the bottom of the page.
The Global stylesheet contains all the general stylings for pages, which are integrated seamlessly.

Table page

Technical Description

The table page contains the table that contains information about the endangered marine species.
<table> element is used to initiate the table. <tr> element is used to add a table row. <thead> element is used to add the elements in the table head. <th> elements holds the column headings of the table. rowspan attribute is used to merge rows into the a single cell in a column.

Accessibility

Tooltips are added to the column headers. When hovered on them the tooltips appears, describing the headers further.

Challenges and Lessons Learned

The main challenge i faced was positioning the elements of the pages while not messing with the layout. Also the Z-index of the elements plays a crucial role when showing the elements on the web page. The z-indices has to be managed with care, with the other elements in mind.
The Header and Footer are copied into all the pages of the site. Therefore, all the pages were viewd manually by me for any defects in those elements. the navigation bar was tested and ensured of proper functining.
The global stylesheet had to be imported to each additional stylesheet that are responsible of layouts of the other pages. I had to make sure that the stylings of the global stylesheet are not overtaken by other stylesheets, making sure the stylings are consistant throughout the whole web site.

Compliance

I made sure the Volunteer page follows Janet rules by avoiding copyright infringement and avoiding the collection of personal data. I have maintained the accessibility by using proper tags and adding alternate text for images. The pages does not contain any offensive content. I admit it is not fully accessible yet but it will be resolved soon.
