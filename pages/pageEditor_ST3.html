Feedback Page

Technical Description

I have used fieldset and legend tags to make the two separate sections of the feedback area. I have used input tags with the type as text to get the users name, email and home addresses and also added a JavaScript to create a custom email validation message using the script tag. I have styled the webpage using CSS I used classes instead of ids' for the ease of applying styling. The use of radio buttons allowed me to create the 5-star rating system of the page and i have used CSS hover effects to make them smooth as possible. Here I have used CSS to make the rating system work as intended when the user presses any star all the stars behind the pressed star will also display gold. Finally I have used textarea tag to get the useres feedback and also used a button tag to create a submit button with the global CSS styling for the button.

Accessibility

I have ensured that all form element do have associated labels for the ease of using screen readers. And also i have used red CSS style for the error message to be visible clearly.

Team Page

Technical Description

I have used HTML div and section tags to make a main container for all the student details and the images. I used another div tag to hold the individual details of the students for each image.The details are in another div tag that is used as an overlay for the details of the student to appear when the user hovers over the image. I used CSS hover effects to make the overlay appear only when the user take their cursor over the student's image.

Accessibility

I have ensured that all the images have alt texts to appear if the image fails to load.
